release の zip をブラウザの拡張機能ページのパッケージ化されていない拡張機能を読み込むで読み込む

スマホでも使えるやつでは使える

スマホでは範囲選択（文字長押し）したあともう一回適当にクリックしたらアイコンが出てくると思います
